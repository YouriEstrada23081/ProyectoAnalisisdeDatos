# ProyectoAnalisisdeDatos
Proyecto analisis de datos problematica: Predicción 6 horas antes de un evento de sepsis en sala de UCI

El dataset implementado supera las capidades de subida de datos que presenta github, por lo que se deja a disposición el link original del reto. https://www.kaggle.com/datasets/salikhussaini49/prediction-of-sepsis/data
Esta es de la plataforma Kaggle en donde es necesrio crearse una cuenta para poder descargar la base de datos.
